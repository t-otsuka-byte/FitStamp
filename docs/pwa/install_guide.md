# スマホへのアプリインストール手順

このアプリ「FitStamp」は、PWA (Progressive Web App) という技術を使っています。
App Store や Google Play からではなく、**ブラウザから直接ホーム画面に追加** することでインストールできます。

## 📱 準備: スマホでアプリを開く

まず、あなたのパソコンとスマートフォンが **同じ Wi-Fi** に繋がっていることを確認してください。

次に、スマホのブラウザ (Safari や Chrome) のアドレスバーに、以下の数字を入力してアクセスしてください。

```
http://192.168.86.25:3000
```

> **注意**: ページが表示されない場合は、PC側のファイアウォール設定を確認するか、PCとスマホが同じWi-Fiネットワーク内にあるか再確認してください。

---

## 🍎 iPhone (iOS) の場合

iPhone の標準ブラウザ **Safari** を使って手順を進めてください。

1. **共有ボタンをタップ**
   画面下の中央にある、四角形から上矢印が出ているアイコン（共有アイコン）をタップします。
   
2. **「ホーム画面に追加」を探す**
   出てきたメニューを少し下にスクロールすると、**「ホーム画面に追加」** という項目があります。これをタップします。
   
3. **「追加」をタップ**
   右上の **「追加」** ボタンをタップします。

🎉 **完了！**  
ホーム画面にオレンジ色の「FitStamp」アイコンが追加されました。タップして起動してみましょう。

---

## 🤖 Android の場合

Android の標準ブラウザ **Chrome** を使って手順を進めてください。

1. **メニューを開く**
   画面右上の **3つの点（︙）** アイコンをタップします。

2. **「アプリをインストール」または「ホーム画面に追加」**
   メニューの中から **「アプリをインストール」** という項目を探してタップします。
   （※機種によっては「ホーム画面に追加」と表示される場合もあります）

3. **確認して追加**
   確認画面が出たら、**「インストール」** または **「追加」** をタップします。

🎉 **完了！**  
ホーム画面にアプリが追加されました。アプリ一覧からも見つけることができます。

---

## ✨ アプリとして使う

ホーム画面に追加されたアイコンから起動すると、画面上部のアドレスバーや下のメニューバーが消えて、**全画面表示** になります。
これで本物のアプリと同じように、毎日の運動記録を付けることができます！
