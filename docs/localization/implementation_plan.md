# 実装計画: 日本語化

## 目標
アプリケーションのユーザーインターフェースおよびメタデータを完全に日本語化し、日本のユーザーにとって使いやすい状態にする。

## 変更内容
- **メタデータ**: サイトタイトル、説明文、アプリ名（PWA用）を日本語に変更。
- **UI コンポーネント**:
  - `Calendar`: `date-fns` の `ja` ロケールを使用し、月名（yyyy年M月）と曜日（日月火...）を日本語化。
  - `Header`: アイコンとアプリ名（FitStamp）はそのまま維持（ブランド名として）。
  - `Footer`: コピーライトの "BookStore" を "FitStamp" に修正し、リンクを「プライバシーポリシー」「利用規約」に変更。
  - `Home`: 英語のキャッチコピーを親しみやすい日本語に翻訳。

## 検証計画
- `npm run dev` で起動し、ブラウザで各画面の表示を確認する。
