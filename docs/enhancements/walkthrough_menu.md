# 検証レポート: ハンバーガーメニュー追加

## 実装内容
画面をすっきりさせるために、称号確認やシェア機能をハンバーガーメニュー（サイドバー）に移動しました。

### 1. 🍔 ハンバーガーメニュー
- **場所**: 画面右上のヘッダー内。
- **動作**: タップすると右側からメニューがスライドインします。
- **内容**: 
    - 現在の称号とランクアップ状況
    - Slack/LINE シェアボタン

### 2. 🏠 状態管理のリファクタリング
- カレンダーとメニューでデータを共有するために `ExerciseContext` を導入しました。
- データの不整合が起きないよう、アプリケーション全体で状態を一元管理しています。

## 検証方法
1. **メニュー開閉**
   - 右上の「≡」アイコンをタップしてメニューが開くか。
   - 「×」アイコンまたは背景タップで閉じるか。
2. **データ同期**
   - カレンダーでスタンプを押した直後にメニューを開き、TOTAL回数やランクが更新されているか。
