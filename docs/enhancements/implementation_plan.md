# 実装計画: 機能強化 (紙吹雪・ストリーク・サマリー)

## 目標
ユーザーのモチベーション向上につながる以下の3つの機能を実装する。
1. スタンプ時の紙吹雪演出
2. 連続記録（ストリーク）表示
3. 月間サマリー表示

## 変更内容

### 1. 紙吹雪演出
- **ライブラリ**: `canvas-confetti` とその型定義 `@types/canvas-confetti` をインストール。
- **実装場所**: `Calendar.tsx`
- **挙動**: 日付をクリックしてスタンプが**ON**になった瞬間、画面中央から紙吹雪を飛ばす。

### 2. 連続記録（ストリーク）
- **ロジック**: `stampedDates`（日付文字列の配列）を解析し、**今日（または昨日）から遡って連続している日数**を計算する。
- **表示**: カレンダーの上部またはメインレイアウトに「🔥 5日連続達成中！」のように目立つように表示。

### 3. 月間サマリー
- **ロジック**: 現在表示している「月」に含まれるスタンプの数をカウントする。
- **表示**: カレンダーの下部に「今月の運動: 12回」のように表示。

## 検証計画
- スタンプを押したときに紙吹雪が出るか。
- 毎日スタンプを押したデータを作り、ストリーク数が正しく増えるか。
- 1日空けた場合にストリークが途切れるか。
- 月を切り替えたときに、その月の集計が正しく表示されるか。
